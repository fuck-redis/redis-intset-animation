## 技术实现细节

### 编码升级机制
```typescript
// 自动检测值范围并升级编码
if (!fitsEncoding(value, currentEncoding)) {
  encoding = getNextEncoding(currentEncoding);
  // 迁移现有数据到新编码
  migrateToNewEncoding(encoding);
}
```

### 二分查找算法
- 时间复杂度：O(log n)
- 返回查找路径用于动画展示
- 未找到时返回应插入位置

### 内存布局可视化
- 每个元素显示：值、十六进制、字节数、索引
- 根据编码类型动态调整元素宽度
- 实时计算总内存占用

### 动画系统
- 基于步骤的动画序列
- 可调节播放速度（0.5x ~ 2.0x）
- 支持插入、查找、删除、升级的完整动画
